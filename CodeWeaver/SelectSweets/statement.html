In this festive season, Chef have received a lot of orders of sweets. One day, while packing sweets, he counted number of different ways he can pack sweets in a box. Two ways are different, if there exist a sweet for which their count in resulting boxes are different. In other words two box are same if they contain same number of all types of sweets. Box has a property that they can contain only a fixed number of sweets, neither less nor more.

If chef has three types for sweets S1, S2, S3 and three boxes (B1, B2, B3). He has to pack 6 sweets in each boxes. If he pack sweets in a way that box has configuration B1{S1 = 1, S2 = 0, S3 = 5}, B2(S1 = 1, S_ = 3, S3 = 2) and B3(S1 = 1, S2 = 0, S3 = 5), then Box B1 and B2 are said to have different configuration, while Box B1 and B3 are same.

But wait, there are some sweets of which customers are very fond of. So they gave you a list of such sweets and the minimum number for each sweet they want in their box. If a sweet is not in this list, then chef can pack any number (including zero) of this type of sweet. Now help chef in count different number of way he can pack sweets in a box. Chef has infinite supply of each types of sweets.

Input:
	First line of input contain three integers, N M K, number of sweets to be packed in a box, number of types of sweets chef had and length of sweet's list which customer gives to him respectevely. Then follows K lines. In each lines there are two integers, Ai Xi, type of sweet and minimum number of sweet he has to put in each box (1 <= i <= M).

Output:
	Find number of different ways chef can pack sweets in a box. As this number can be very large print answer modulo 1000000007 (10^9 + 7).

Constraints:
	1 <= N <= 1000000
	1 <= M <= 1000000
	1 <= K <= M

Sample Input #0
4 2 1
1 2

Sample Output #0
3


Sample Input #1
5 3 2
1 2
2 1

Sample Output #1
6


Sample Input #2
5 1 1
3 6

Sample Output #2
0


Explanation for Sample Input #0
Chef has to pack 4 sweets in a box, and he have two types of sweets (S1, S2). Customer has mentioned that the box must have at least 2 sweet of sweet type #1. Then there are 3 number of ways of packing sweets, represented by (S1 = 2, S2 = 2), (S1 = 3, S2 = 1), (S1 = 4, S2 = 0).

Explanation for Sample Input #2
Chef has to pack 5 sweets in a box, while according to list he has to pack at least 6 sweets of sweet type 3. And it is not possible.
