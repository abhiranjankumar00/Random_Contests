Problem
You are given a mathematical expression and you have to evaluate it to get the result. Since you are in first year you only know 4 operators (+,-,*,/) and you don't know BODMAS as well, so you evaluate the expression from left to right.You also know numbers from (1-9) only. The result will fit in 63 bit memory. The result shold be of 8 decimal point precision.
First line of input is number of test cases T (1<=T<=100). Each line of test case will consist of mathematical expression. Output T lines with result of each expression. The result shold be of 8 decimal point precision.

Constraints :
length of string : 1<=|s|<=10000

Input :
2
1+2*3
3/4

Output :
9.00000000
0.75000000

Time limit : 1 sec
