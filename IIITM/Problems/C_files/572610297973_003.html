Problem
You just came to know about odd and even numbers. Your friend tried to confuse you with odd-even numbers. He asked "Given a set of numbers, say YES if number of odd numbers is even and number of even numbers is odd, otherwise say NO". Can you solve this problem using programming?
First line of input is number of test cases T (1<=T<=100). Each test case will consist of 2 lines. First line will consist of N number of numbers. Next line will have N numbers. Output T lines with YES or NO.

Constraints :
1<=N<=1000

Input :
2
7
1 2 3 4 5 6 7
6
1 2 3 4 5 6

Output :
YES
NO

Time limit : 1 sec
