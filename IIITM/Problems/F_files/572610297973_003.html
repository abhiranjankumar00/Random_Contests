Problem
You are new to mathematics and just came to know about circles and squares. You are given description of circle or square and a point. You need to find if the point lies inside or on the figure. Print NO if it lies outside the figure.NOTE: Sides of square are parellel to coordinate axes.
First line of input is number of test cases T (1<=T<=100). Each test case will consist of 1 line. If the figure is circle the input will have its first integer '0' followed be description of circle (x-x coordinate , y-y coordinate of centre, r-radius), then (X',Y') integer coordinates of point. If the figure is square the input will have its first integer '1' followed be description of square (Lower left vertex of square (lx,ly), and upper right corner of square(hx,hy)), then (X',Y') integer coordinates of point. Output T lines with YES/NO.

Input :
2
0 0 0 5 1 1
1 1 1 3 3 2 2
Output :
YES
YES

Time limit : 1 sec
