Problem
In new world fibonacci series is defined by
f(n) = f(n-2) + f(n-3)
f(0) = 1; f(1) = 2; f(2) = 3;
Given N find Nth fibonacci number. First line of input is number of test cases T (1<=T<=10000). Each test case will consist of N(1<=N<=1000000). Output T lines with result modulo 10000007.

Input :
2
2
4
Output :
3
5

Time limit : 1 sec
